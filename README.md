# Dynamic JS Page (Header + Cards)

This project is a simple dynamic webpage built entirely using **JavaScript DOM**, without writing HTML structure manually.  
The page includes:

- A dynamic header with a title and navigation bar
- Interactive hover animations
- Auto-generated cards using loops
- Basic styling added from JS (for practice)
